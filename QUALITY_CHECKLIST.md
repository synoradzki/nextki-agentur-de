# Qualitäts-Checklist – Self-Review

Vor jedem Commit prüft der Loop den eigenen Content anhand dieser Kriterien:

## Content-Qualität
- [ ] Enthält der Text eine persönliche Perspektive, nicht nur generische Aussagen?
- [ ] Gibt es konkrete Beispiele, Zahlen oder Datenquellen?
- [ ] Variiert die Satzlänge natürlich (kurz/lang gemischt)?
- [ ] Hat der Text einen erkennbaren Standpunkt?
- [ ] Würde ein Experte mit Fachwissen das so schreiben?
- [ ] Gibt es echte Quellenangaben (nicht nur "Studien zeigen")?

## SEO-Qualität
- [ ] Title-Tag enthält das Ziel-Keyword und ist ≤ 60 Zeichen?
- [ ] Meta-Description ist überzeugend und ≤ 155 Zeichen?
- [ ] H1 enthält das Keyword und ist einzigartig auf der Seite?
- [ ] Interne Verlinkung zu mindestens 2 anderen Seiten?
- [ ] Bilder haben Alt-Text (wenn vorhanden)?
- [ ] Schema Markup ist korrekt (JSON-LD validiert)?

## Technische Qualität
- [ ] Astro Build läuft fehlerfrei?
- [ ] Keine toten Links eingebaut?
- [ ] Seite ist mobilfreundlich (responsive)?

## Sicherheits-Leitplanken
- [ ] Maximal 30% eines bestehenden Textes geändert?
- [ ] Keine geschützten Elemente modifiziert (URLs, Impressum, Kontaktdaten)?
- [ ] Mindestens 7 Tage seit letzter Optimierung vergangen?
- [ ] DSGVO-konform (keine personenbezogenen Daten ohne Grundlage)?
